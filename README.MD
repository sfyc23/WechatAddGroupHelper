# 基于 itchat 的微信加群助手

主要实现的功能有：

1. 通过关键字自动同意加好友。
2. 通过关键字自动邀请好友入群，可添加多个备用群。
3. 微信断线后发送提醒邮件。
4. 腾讯 AI 自动回复。

**itchat 依赖于网页版微信进行开发，所以首先得确定你的微信能登录网页版微信**：<https://wx.qq.com/>。
